# TenderAI – AI Powered eProcurement & Tender Evaluation Platform

AI-Based Tender Evaluation System is an intelligent e-procurement platform designed to streamline and automate the tender evaluation process for government departments and public sector organizations. Traditional tender evaluation involves extensive manual document verification, eligibility checking, compliance validation, and comparative analysis, which is time-consuming and prone to human errors.

This system leverages Artificial Intelligence to analyze tender documents, evaluate bidder eligibility, verify compliance requirements, and generate transparent evaluation results in a significantly shorter time. The platform provides a centralized environment where procurement authorities can manage tenders, review submissions, and make data-driven decisions efficiently.

The primary objective of the project is to improve transparency, reduce evaluation time, minimize manual effort, and ensure fair tender selection through AI-assisted decision-making.

- Google Chrome 70+

System requirements:
- Minimum resolution: **1280 × 720**
- JavaScript enabled
- Cookies enabled

### Digital Signature Certificate (DSC)
A valid **Class III DSC** is required for bid submission.

Requirements:
- Licensed Certificate Authority (CA)
- Installed USB token drivers
- Proper DSC configuration

### Java Requirement
Java **JRE 1.8 or higher** is required for digital signing utilities.

Configuration:
- Install Java Runtime Environment
- Add eProcure URL to Java Exception Site List

### Hassle-Free Bid Submission Guidelines
Recommended practices:
- Submit bids at least **2 hours before the deadline**
- Keep DSC connected during submission
- Do not close the browser during uploads

# Tech Stack

TenderAI is built using modern technologies to provide a smooth procurement and tender evaluation experience.

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Spring Boot, REST APIs  
- **Database:** MySQL  
- **AI Layer:** Document Analysis, Eligibility Evaluation  

---

# Project Architecture

The application follows a layered architecture for efficient processing and scalability.

```text
Frontend (React + Tailwind)
            ↓
      REST API Layer
            ↓
 Spring Boot Backend
      ↓           ↓
AI Engine     MySQL Database
```

The frontend handles user interactions, the backend processes requests, the AI engine evaluates documents, and MySQL stores application data.

# Installation & Setup

### Prerequisites
- Java 17+
- Node.js
- MySQL
- Maven

### Clone Project

```bash
git clone <repository-url>
cd TenderAI
```

### Run Frontend

```bash
npm install
npm run dev
```

### Run Backend

```bash
mvn spring-boot:run
```

Application runs locally after setup.
