# TenderAI – AI Powered eProcurement & Tender Evaluation Platform

TenderAI is an AI-driven eProcurement platform designed to simplify tender discovery, bidder evaluation, and procurement decision-making. The platform enables organizations to browse tenders, upload tender and bidder documents, and generate AI-based eligibility evaluations within seconds.

# 5. AI Tender Evaluation Engine

The AI Evaluation Engine is the core feature of TenderAI and automates bidder eligibility analysis.

### Workflow

#### Step 1 – Upload Tender Document
Upload the official tender notice (NIT/RFP).

Supported formats:
- PDF
- DOC
- DOCX

#### Step 2 – Upload Bidder Documents
Upload bidder submissions and supporting documents.

Supported formats:
- PDF
- DOC
- DOCX

#### Step 3 – Run AI Evaluation
After document upload, the system automatically processes:
- Eligibility verification
- Document analysis
- Bid compliance validation
- Evaluation scoring

### AI Insights
- Evaluation Count
- Average Processing Time
- Accuracy Metrics

### Purpose
Reduce manual procurement effort and generate explainable eligibility decisions.

# 9. Help & Site Compatibility Page

The Help & Site Compatibility page provides users with technical requirements, setup guidance, and support resources to ensure a smooth and hassle-free tender submission process.

### Features
- Browser compatibility recommendations.
- Digital Signature Certificate (DSC) setup support.
- Java environment configuration guidance.
- Bid submission best practices.
- Contractor help and support resources.

### Recommended Browsers
The platform is supported on:
- Mozilla Firefox 60+
- Google Chrome 70+
- Microsoft Edge 90+

System requirements:
- Minimum resolution: **1280 × 720**
- JavaScript enabled
- Cookies enabled

### Digital Signature Certificate (DSC)
A valid **Class III DSC** is required for bid submission.

Requirements:
- Licensed Certificate Authority (CA)
- Installed USB token drivers
- Proper DSC configuration

### Java Requirement
Java **JRE 1.8 or higher** is required for digital signing utilities.

Configuration:
- Install Java Runtime Environment
- Add eProcure URL to Java Exception Site List

### Hassle-Free Bid Submission Guidelines
Recommended practices:
- Submit bids at least **2 hours before the deadline**
- Keep DSC connected during submission
- Do not close the browser during uploads

### Help for Contractors
Support resources include:
- Bidder manuals
- Video tutorials
- Frequently Asked Questions (FAQs)
- Real-time support assistance

### Purpose
This section helps users complete system setup correctly and minimize technical issues during procurement participation.

### User Actions
- Verify browser compatibility
- Configure DSC and Java
- Access help resources
- Follow submission guidelines
